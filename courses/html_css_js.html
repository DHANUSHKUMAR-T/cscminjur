HTML / CSS / JavaScript

HTML, CSS & JavaScript Basics
Duration: 2 Months (Approx.)

Overview

This course is designed for beginners who want to start their career in Web Development. Students will learn how to create modern and responsive websites using HTML, CSS, and JavaScript. It is a foundation course for advanced technologies like React, Angular, Node.js, and Full Stack Development.

Objective

Upon completion of this course, the participant will be able to design and develop static and interactive web pages. Students will gain hands-on experience in building websites, creating forms, applying styling, and writing JavaScript logic.

Target Audience

School Students / Diploma / Graduates / Engineering Students / Beginners

Prerequisites

No prior coding knowledge is required.

Course Coverage

HTML5

  • Introduction to Web Development
  • HTML Structure and Basic Tags
  • Headings, Paragraphs, Links, Images
  • Lists and Tables
  • Forms and Input Elements
  • Semantic HTML Elements
  • Audio, Video and Multimedia

CSS3

  • CSS Syntax
  • Selectors (Class, ID, Tag)
  • Colors, Backgrounds, Borders
  • Box Model
  • Margin, Padding, Display
  • Positioning (Relative, Absolute, Fixed)
  • Flexbox Layout
  • Responsive Design Basics
  • Transitions and Animations

JavaScript

  • Introduction to JavaScript
  • Variables and Data Types
  • Operators and Expressions
  • Conditional Statements (if, else)
  • Loops (for, while)
  • Functions and Return Values
  • Arrays and Objects
  • DOM Manipulation
  • Event Handling (click, submit)
  • Basic Form Validation

Project

Students will develop mini projects such as Portfolio Website, Calculator, To-Do List, Registration Form, Landing Page, and simple interactive websites.
