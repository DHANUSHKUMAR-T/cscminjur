HDCA (Programming)

Honours Diploma in Computer Application (Programming)
Duration: 1 Year

Students must complete two projects using C/C++ and Python.

Objective

Upon completion of this course, the participant acquires the advantage in choosing his / her career as an Application Programmer. Students will learn to create applications using object oriented programming concepts such as inheritance, polymorphism and virtual functions.

Prerequisites

No prior computer knowledge is required.

Overview

This course has been tailor made as per the software industry requirement. It is a stepping stone for those who aspire to be a software professional.

Target Audience

School final students / Graduates / Students pursuing a career in Computer.
