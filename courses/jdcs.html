JDCS

Junior Diploma in Computer Skills
Duration: 2 Months (Approx.)

Overview

JDCS (Junior Diploma in Computer Skills) is a special course designed for school students and beginners. This course helps kids learn basic computer knowledge, improve typing skills, and build confidence in using computers for studies and daily life.

Objective

The objective of this course is to provide basic computer skills to students and help them understand how to use a computer effectively. Students will learn simple software tools, internet usage, and basic MS Office applications.

Target Audience

Kids / School Students / Beginners

Prerequisites

No prior computer knowledge is required.

Course Coverage

Computer Fundamentals

  • Introduction to Computer
  • Parts of Computer (CPU, Monitor, Keyboard, Mouse)
  • Hardware and Software Basics
  • Operating System Basics
  • Using Keyboard and Mouse

Windows Basics

  • Desktop, Icons, Taskbar
  • File and Folder Creation
  • Copy, Paste, Delete Files
  • Using Windows Explorer

MS Paint

  • Drawing and Coloring
  • Shapes and Tools
  • Creative Painting Activities

Typing Practice

  • Keyboard Practice
  • Basic Typing Speed Improvement
  • Simple Words and Sentences Practice

MS Word Basics

  • Typing Documents
  • Font Style and Size
  • Paragraph Alignment
  • Inserting Images
  • Saving and Printing Documents

MS PowerPoint Basics

  • Creating Slides
  • Adding Text and Images
  • Design Themes
  • Basic Slide Animations

Internet Basics

  • Introduction to Internet
  • Using Browser (Chrome)
  • Searching Information in Google
  • Basic Online Safety

Project

Students will complete fun mini projects such as PowerPoint Presentation, Typing Practice Sheets, Drawing in Paint, Simple Word Document Projects, and Internet Search Activities.
