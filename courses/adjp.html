ADJP

Advanced Diploma in Java Programming
Duration: 4 Months

Overview

This course enables you with skills to do effective programming for internet applications. It teaches you advanced programming techniques for the internet.

Objective

Upon completion of this course, the participant will be in a position to create components for the internet and integrate them.

Target Audience

Diploma / Graduates / Engineering Students

Prerequisites

Basic knowledge of Computing.
